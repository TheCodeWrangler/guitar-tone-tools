# Mermaid's Ocean

A small underwater game: swim as a mermaid and meet ocean friends.

## How to play

- **Desktop:** Use **arrow keys** to swim.
- **Mobile:** **Drag** the mermaid to move.
- Swim near a friend (turtle, fish, octopus, seahorse, starfish) to meet them and read their greeting. The counter shows how many friends you’ve met.

## Run locally

Open `index.html` in a browser, or serve the folder:

```bash
# Python
python3 -m http.server 8000

# Node (npx)
npx serve .
```

Then open `http://localhost:8000` (or the port shown).
